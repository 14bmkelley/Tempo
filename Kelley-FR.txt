1) The system shall collect the user’s Google account information and connect to the Google API when a user signs in.
2) The system shall find and load another user’s calendar information when a user performs a search on that user.
3) The system shall create a group when a user requests to form a group.
4) The system shall match empty spaces in the calendars of a group’s members when a member requests to schedule a meeting time.
5) The system shall update each user’s calendar with a new event when a group confirms a meeting.

Dictionary
Group: A number of users who need to compare their calendars for a meeting time.
Member: A user who is part of a group.
Meeting: A time set aside to meet. Can contain other information, like location or topic.
